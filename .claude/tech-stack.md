## 技術棧

**主要語言：** Python 3.10+

| 用途 | 工具 |
|------|------|
| 數據處理 | pandas / numpy |
| 本地資料庫 | sqlite3 |
| 視覺化 | matplotlib |
| 機器學習 | sklearn |
| 快速原型 / 個人工具 | streamlit |
| 通用介面 / 對外部署 | Vanilla HTML/JS（CDN library 可接受，無打包工具）|
| 後端 API | Flask / FastAPI（視需求整合）|

**領域：** 製造業（MES、製程管理、品質管理）

**原則：**
- 外部工具或框架需對專案有明確效益才引入
- 優先維持自己可掌控的程式碼範圍
